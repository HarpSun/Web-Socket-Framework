# Web-Socket-Framework
基于TCP连接和Socket开发的Web Framework

## 演示
![demo](https://github.com/HarpSun/Web-Socket-Framework/blob/master/screencast/demo.gif)


## 结构

| 文件/目录  |   说明   |    分层    |
| ---------- | -------- | ---------- |
| server.py  | 程序入口 | -          |
| models/    | 数据模型 | Model      |
| routes.py    | 转发路由 | Controller |
| templates/ | 前端模板 | View       |
| static/    | 静态资源 | -          |
